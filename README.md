# ts-playground

## 项目初始化

```JavaScript
> git init
> yarn init
> tsc init
> yarn add -D husky
> yarn add -D prettier
> commitizen init cz-conventional-changelog --save --save-exact
```

在`package.json`写脚本,添加 `.gitignore`,`.prettierrc`

运行 `yarn log` 初始化 `CHANGELOG.md`

修改 `tsconfig.json` 中的 rootDir / outDir

编写示例代码

编译`cmd+shift+b`或者`yarn build`

打断点 debug 运行
